# 💱 Conversor de Moedas — Django

## 📖 Sobre o projeto

Este é um **mini projeto de estudo** desenvolvido em grupo com o objetivo de praticar conceitos de **desenvolvimento web com Django**.

O sistema permite ao usuário **converter valores entre diferentes moedas** e **registrar o histórico das conversões** no banco de dados.

---

## 🎯 Objetivos de aprendizado

- Criação de **views** no Django
- Uso de **templates HTML**
- Construção de **formulários**
- Integração entre **backend** e **frontend**
- Persistência de dados em banco **PostgreSQL**
- Organização da estrutura de um **projeto Django**
- Manipulação de requisições **GET**
